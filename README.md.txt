# Dividend Calculator

This app helps calculate dividends from unit trust investments.

## Download the APK

You can download the APK file [here](https://github.com/Nabilah333/DividendCalc/raw/refs/heads/master/apk/app-debug.apk).

After downloading, open the APK file on your Android device to install the app.  
You may need to enable installation from unknown sources in your device settings.
